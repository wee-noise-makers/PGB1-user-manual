# Copy & Paste

The PGB-1 allows you to copy and paste four types of elements: Song Parts, Tracks, Patterns, and Steps.

## Starting Copy/Paste

1. **Hold** the ++cpy-fx++ button (keep it held throughout)
2. Press the mode button for what you want to copy:
   - ++song++ - Copy song parts
   - ++track++ - Copy tracks
   - ++pattern++ - Copy patterns
   - ++step++ - Copy steps
3. Enter the source ("From") address
4. Enter the destination ("To") address
5. Release ++cpy-fx++

## Understanding Addresses

Different elements have different address formats:

| Element | Address Format |
|---------|---------------|
| Song Part | Part number (1-12) |
| Track | Track number (1-16) |
| Pattern | Track + Pattern (e.g., Track 1, Pattern 3) |
| Step | Track + Pattern + Step (e.g., Track 1, Pattern 3, Step 5) |

## Copy Types

### Copying Song Parts

Copies all song part settings to another part.

**Procedure:**
1. Hold ++cpy-fx++ + press ++song++
2. Press source part (++1++ to ++12++)
3. Press destination part (++1++ to ++12++)

### Copying Tracks

Copies all patterns from one track to another.

!!! note "Track Settings Not Copied"
    Copying tracks copies pattern data only, not track settings (engine, effects, etc.).

**Procedure:**
1. Hold ++cpy-fx++ + press ++track++
2. Press source track (++1++ to ++16++)
3. Press destination track (++1++ to ++16++)

### Copying Patterns

Copies all steps and pattern settings from one pattern to another.

!!! note "Pattern Settings Included"
    Copying patterns includes pattern settings (length, link).

**Procedure:**
1. Hold ++cpy-fx++ + press ++pattern++
2. Screen shows "From" with Track and Pattern to enter
3. Press ++track++ if needed to change track
4. Press source track number
5. Press source pattern number
6. Press ++track++ if needed to change destination track
7. Press destination pattern number

### Copying Steps

Copies a single step's settings to another step.

**Procedure:**
1. Hold ++cpy-fx++ + press ++step++
2. Enter source: Track, Pattern, Step numbers
3. Enter destination: Track, Pattern, Step numbers

## Address Entry Details

### Current Position

When copying patterns or steps, the address starts with your current track and pattern. Blinking question marks indicate what to enter next.

### Changing Address Components

For patterns and steps, you may need to change the track or pattern before entering the final number:

- Press ++track++ to enter/change the track number
- Press ++pattern++ to enter/change the pattern number

### Multiple Destinations

After the first paste, the "To" address shows blinking question marks again. You can immediately paste to another destination without re-selecting the source.

## Examples

### Copy Step 2 to Steps 4, 6, and 8 (Same Pattern)

1. Hold ++cpy-fx++ + press ++step++
2. Press ++2++ (source step)
3. Press ++4++ (first destination)
4. Press ++6++ (second destination)
5. Press ++8++ (third destination)
6. Release ++cpy-fx++

### Copy Step from Track 7, Pattern 5, Step 2 to Track 1, Pattern 3, Step 9

1. Hold ++cpy-fx++ + press ++step++
2. Press ++track++, then ++7++ (source track)
3. Press ++5++ (source pattern, or press ++pattern++ then ++5++)
4. Press ++2++ (source step)
5. Press ++track++, then ++1++ (dest track)
6. Press ++3++ (dest pattern)
7. Press ++9++ (dest step)
8. Release ++cpy-fx++

### Copy Pattern 1 to Patterns 2, 3, 4 (Same Track)

1. Hold ++cpy-fx++ + press ++pattern++
2. Press ++1++ (source pattern)
3. Press ++2++ (first destination)
4. Press ++3++ (second destination)
5. Press ++4++ (third destination)
6. Release ++cpy-fx++

## Tips

### Building Patterns Quickly

1. Create a basic pattern on one track
2. Copy it to other tracks
3. Modify each copy for variation

### Creating Variations

1. Copy a pattern to another slot
2. Make small changes to the copy
3. Use both in different song parts

### Step Templates

1. Configure a step with desired settings
2. Copy to multiple steps quickly
3. Adjust notes/conditions as needed

### Workflow

- Copy before modifying (backup your work)
- Use multiple paste for repetitive steps
- Plan your track and pattern organization
