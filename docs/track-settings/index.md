# Track Settings

Track Settings is where you shape your sounds and configure how each track behaves. This section covers all the parameters available when in Track Mode.

## Accessing Track Settings

1. Press ++track++ to enter Track Mode
2. Press ++1++ to ++16++ to select a track
3. Use ++left++ / ++right++ to navigate between settings pages

## Settings Overview

| Page | Setting | Description |
|------|---------|-------------|
| 1 | Synth Engine | Sound generator selection |
| 2 | Engine Parameters | Sound shaping controls |
| 3 | LFO | Modulation settings |
| 4 | Volume | Track level |
| 5 | Pan | Stereo position |
| 6 | Effect | Insert effect |
| 7 | Octave | Pitch offset |
| 8 | Shuffle | Swing amount |
| 9 | Arpeggiator Mode | Arp pattern |
| 10 | Arpeggiator Notes | Note selection |
| 11 | Track Mode | Internal/MIDI |

- Hold **Cpy/FX** to access quick parameter control with the touch strip
- Preview sounds while editing by pressing track buttons
- Use the touch strip for smooth parameter adjustments
