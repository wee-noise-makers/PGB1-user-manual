# Main Menu

The main menu provides access to system functions, project management, and device settings.

## Accessing the Menu

Press ++menu++ to open the main menu.

Navigate with ++left++ / ++right++ and select with ++a++.

## Menu Items

### Projects

Manage your saved projects:

| Option | Description |
|--------|-------------|
| Save | Save current project to a slot |
| Load | Load a project from storage |
| New | Create a new empty project |

#### Loading a Project

1. Select **Load**
2. Use ++up++ / ++down++ to browse projects
3. Press ++a++ to select
4. Confirm the action

!!! warning "Data Loss"
    Loading a project will erase the current project's patterns, steps, tracks, and song settings.

#### Saving a Project

1. Select **Save**
2. Choose a project slot
3. Confirm to save

#### New Project

Creates an empty project with default settings.

### Inputs

Configure audio input settings. See [Audio Input](../audio-midi/input.md).

| Page | Settings |
|------|----------|
| 1 | Input enables and volume |
| 2 | Input effects |

### Custom Waveform

Draw and edit custom waveforms. See [Custom Waveforms](../custom-waveforms.md).

*Available from firmware 1.2.0*

### Update Mode

Enter firmware update mode. See [Firmware Updates](firmware.md).

!!! warning "Important"
    Entering update mode will save your project and reboot the device.

### System Info

View device information:

- Firmware version
- Hardware revision
- Other system details

Useful for:

- Checking if you need to update
- Support requests
- Troubleshooting

## Navigation Tips

### Quick Access

The menu position is remembered:

- Exit with ++b++ to return to where you were
- Re-enter menu to continue from that position

### Menu Position Indicator

Small dots at the top show:

- How many menu pages exist
- Which page you're currently on
- Arrows indicating more pages left/right

## Tips

- Check System Info to know your firmware version
- Save your project before entering Update Mode
- Use Input settings to configure external audio
- Create new projects to start fresh without loading
