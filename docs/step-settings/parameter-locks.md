# Parameter Locks

Parameter locks allow you to set different synth parameters for individual steps, enabling sound variations within a single pattern.

## What Are Parameter Locks?

Normally, all steps in a pattern use the track's synth settings. Parameter locks let you override specific parameters on a per-step basis.

## Accessing Parameter Locks

1. Enter Step Mode (press ++step++)
2. Select a step
3. Navigate past the standard settings (Condition, Note, Duration, Velocity, Repeat)
4. Additional pages contain parameter locks

## Available Parameter Locks

The available parameters depend on the track type:

### Synth Tracks

- Engine Parameter 1
- Engine Parameter 2
- Engine Parameter 3
- Engine Parameter 4

### Sample Tracks

- **Sample Selection** - Choose which sample to play
- Start Point - Where in the sample to begin playback
- Other sample parameters

## Sample Selection Per Step

This is one of the most powerful uses of parameter locks:

### Playing Different Samples Per Step

1. Select a sample track (7 or 8)
2. Enter Step Mode
3. Select step 1
4. Navigate to the Sample parameter lock
5. Choose a sample (e.g., Kick)
6. Select step 2
7. Choose a different sample (e.g., Snare)
8. Continue for each step

Now one sample track can play an entire drum kit!

### Example: Full Kit on One Track

| Step | Sample |
|------|--------|
| 1, 5, 9, 13 | Kick |
| 3, 7, 11, 15 | Hi-hat |
| 5, 13 | Snare |
| 8 | Clap |

This maximizes your sample track usage.

## Using Multiple Sounds from One Sample

If you record multiple sounds into a single sample:

1. Select a step
2. Set the Start Point parameter lock to the beginning of sound 1
3. Select another step
4. Set Start Point to the beginning of sound 2

This lets you fit multiple sounds in one sample slot.

## Practical Applications

### Drum Variation

1. Create a basic pattern
2. On every 4th snare hit, change a parameter
3. Adds subtle variation to repetitive patterns

### Filter Sweeps

1. Set filter cutoff parameter locks on consecutive steps
2. Each step has a slightly higher cutoff
3. Creates a sweep effect across the pattern

### Accent Patterns

1. Set certain steps with higher resonance
2. Creates accented hits without changing velocity

### Sound Evolution

1. Gradually change parameters across a pattern
2. First half: one sound character
3. Second half: evolved sound

## Tips

- Parameter locks are saved with the pattern
- Copying a step copies its parameter locks
- Use sparingly to avoid chaos - subtle changes work best
- Sample selection locks are essential for efficient sample track use
- Remember: parameter locks override track settings only for that step
