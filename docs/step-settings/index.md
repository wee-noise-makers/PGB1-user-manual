# Step Settings

Step Settings let you configure individual steps in a pattern, providing detailed control over timing, pitch, velocity, and sound parameters.

## Accessing Step Settings

1. Press ++step++ to enter Step Mode
2. Press ++1++ to ++16++ to select a step
3. Use ++left++ / ++right++ to navigate settings pages

## Settings Overview

| Page | Setting | Description |
|------|---------|-------------|
| 1 | Condition | When the step triggers |
| 2 | Note | What note to play |
| 3 | Octave | Octave offset |
| 4 | Duration | Note length |
| 5 | Velocity | Note intensity |
| 6 | Repeat | Retrigger settings |
| 7+ | Parameter Locks | Per-step sound parameters |

## In This Section

- [Trigger Conditions](conditions.md) - Control when steps play
- [Notes & Chords](notes.md) - Set pitches and chord notes
- [Parameter Locks](parameter-locks.md) - Per-step sound changes

## Quick Tips

- Use velocity for groove and dynamics
- Combine conditions with copy/paste for variation
- Parameter locks can change samples per step
