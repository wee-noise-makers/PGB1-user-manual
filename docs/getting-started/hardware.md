# Hardware Layout

## Front Panel

### Keyboard Section

The 16 numbered buttons (++1++ to ++16++) serve multiple purposes depending on the current mode:

| Mode | Button Function |
|------|-----------------|
| Track Mode | Select track 1-16 |
| Step Mode | Select step 1-16 |
| Pattern Mode | Select pattern 1-16 |
| Song Mode | Buttons 1-12: Song Parts, Buttons 13-16: Chord Progressions |
| Edit Mode | Toggle steps on/off in the pattern |

### Mode Buttons

- ++track++ - Enter Track Mode (configure instruments and sounds)
- ++step++ - Enter Step Mode (configure individual step parameters)
- ++pattern++ - Enter Pattern Mode (configure pattern length and linking)
- ++song++ - Enter Song Mode (arrange parts and chord progressions)

### Navigation Buttons

- ++left++ ++right++ - Navigate between settings pages
- ++up++ ++down++ - Change parameter values
- ++a++ - Confirm / Select / Enable
- ++b++ - Back / Cancel / Disable

### Control Buttons

- ++menu++ - Open the main menu
- ++play++ - Start/Stop the sequencer (also used for shortcuts when held)
- ++edit++ - Toggle pattern edit mode / Hold for chromatic keyboard

### Special Button

- ++cpy-fx++ - Hold for Live FX access and copy/paste operations

### Touch Strip

The capacitive touch strip at the bottom of the device allows for:

- Precise parameter value adjustment
- Live performance expression
- Quick value entry

### LEDs

- **Status LED** (top right) - Shows power state and charging status
- **Mode LEDs** - Indicate the current active mode
- **Keyboard LEDs** - Show step states, pattern links, and various information

## Connectors

### Top Edge

- **Headphone/Speaker Output** (3.5mm stereo jack) - Connect headphones or external speakers
- **Line Input** (3.5mm stereo jack) - Connect external audio sources

### Bottom Edge

- **USB Type-C** - For charging and firmware updates
- **Reset Button** (small hole) - Use a paperclip to reset the device
- **Boot Button** (small hole) - For forced firmware update procedure

### Power

- **Power Switch** (side) - Slide left to turn on, right to turn off

## Internal Components

- **Built-in Speaker** - For monitoring without headphones
- **Internal Microphone** - For recording samples
- **Rechargeable Battery** - Provides portable operation
