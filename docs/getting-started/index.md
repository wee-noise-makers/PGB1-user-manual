## Quick Overview

The PGB-1 interface consists of:

- **16 numbered buttons** (1-16) - Used for selecting tracks, steps, patterns, and playing notes
- **Mode buttons** - ++track++ ++step++ ++pattern++ ++song++ to switch between main modes
- **Navigation buttons** - ++left++ ++right++ ++up++ ++down++ ++a++ ++b++ for menu navigation
- **Control buttons** - ++menu++ ++play++ ++edit++ for system functions
- **Touch strip** - For parameter adjustment and live performance
